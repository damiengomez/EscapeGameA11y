Handicap visuel

00:00:00

Description

La navigation sur internet est un défi pour une personne atteint de déficience visuel.
L'expérience en ligne est très dépendante des aides techniques (comme les lecteurs d'écran), et d'un code html adapté s'interfacer correctement aux outils.

Règles

  • Résoudre la prochaine énigme en indiquant les valeurs attendus dans les bons champs de formulaire
  • Cliquez sur les boutons dans un ordre précis : vert, bleu, rouge et ensuite violet pour commencer
  • Attention, en cas d'erreur il faudra tout recommencer !
  • Pour la suivante, souvenez-vous : "Tab" on avance, "shift+tab" on recule...
