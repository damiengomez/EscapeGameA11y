Handicap auditif

Description

La navigation sur internet peut être un défi pour les personnes ayant des déficiences visuel.
Pour elles, l'expérience en ligne peut être très dépendante des aides techniques comme les lecteurs d'écran et d'un code adapté pour faire fonctionner correctement ces outils.

Règles

  • Résoudre la prochaine énigme en indiquant les valeurs attendus dans les bons champs de formulaire
  • Cliquez sur les boutons dans un ordre précis : vert, bleu, rouge et ensuite violet pour commencer
  • Attention, en cas d'erreur il faudra tout recommencer !
